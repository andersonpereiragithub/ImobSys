## Objetivo do Projeto
O **ImobSys** é um sistema de controle e gestão de imóveis, inicialmente desenvolvido como uma aplicação de console. A estrutura do sistema foi planejada para permitir futuras migrações para interfaces web ou desktop, com suporte a persistência de dados em diferentes formatos (iniciando com JSON e preparado para migração para banco de dados).
